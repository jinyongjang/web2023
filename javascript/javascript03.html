01. 선언적 함수

선언적 함수(Declaration function)는 자바스크립트에서 함수를 정의하는 가장 일반적인 방법 중 하나입니다.
함수 이름을 지정하고, 매개 변수를 정의하고, 함수 몸체를 작성하여 함수를 생성합니다. 이 함수를 호출할 때는 함수 이름과 괄호를 사용합니다.
함수의 이름은 "func"이며, 함수 정의는 중괄호({})로 묶여 있습니다. 함수를 호출하려면, 함수 이름 다음에 괄호를 사용하고,
필요한 인수(파라미터)를 괄호 안에 넣습니다. 이 코드에서는 인수가 없는 함수를 호출하고 있습니다.

 {
    function func(){
        document.write("실행되었습니다.");
    }
    func();
}
결과 확인하기
"실행되었습니다."

02. 익명 함수

익명 함수(Anonymous function)는 함수 이름이 없는 함수입니다. 익명 함수는 함수 표현식을 사용하여 정의하며, 변수에 할당하여 사용합니다.
먼저, "func"라는 변수에 함수 표현식을 할당합니다. 이 함수 표현식은 "function()" 키워드를 사용하여 함수를 정의하고, 중괄호({}) 내부에 함수의 코드를 작성합니다.
이후, 변수 이름을 사용하여 함수를 호출합니다.

 {
    const func = function(){
        document.write("실행되었습니다.");
    }
    func();
}
결과 확인하기
"실행되었습니다."

01. 변수 : 데이터 저장

변수는 데이터를 저장하는 저장소입니다.

 {
    var x = 100;            //변수 X에 숫자 100을 저장함
    var y = 200;            //변수 y에 숫자 200을 저장함
    var z = "javascript";   //변수 z에 문자 "javascript"를 저장함

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
