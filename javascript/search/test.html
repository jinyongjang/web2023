• caption-side : 속성은 표(table) 요소에서 캡션(caption) 요소의 위치를 지정하는 데 사용됩니다.
  • caret-color : 텍스트 입력 시 표시되는 캐럿(caret, 입력 포인터)의 색상을 지정하는 속성입니다.
  • clear : 속성은 부모 요소와 자식 요소 간의 레이아웃 배치 관계를 설정하는 CSS 속성 중 하나입니다.
  • clip : 속성은 요소의 표시 영역을 지정하는 데 사용됩니다.
  • color : 속성은 텍스트의 색상을 지정하는 속성입니다.
  • column-count : 속성은 다단(column) 레이아웃 속성 중 하나로, 요소를 여러 개의 열(column)로 나누는 역할을 합니다.
  • column-fill : 속성은 다단(column) 레이아웃 속성 중 하나로, 다단 요소의 열(column)을 어떻게 채울지 결정하는 역할을 합니다.
  • column-gap : 속성은 다단(column) 레이아웃 속성 중 하나로, 다단 요소의 열(column) 사이의 간격을 지정하는 역할을 합니다.
  • column-rule : 속성은 다단 레이아웃(column layout)을 만들 때, 각각의 컬럼(column) 사이에 선을 추가하는 데 사용됩니다.
  • column-rule-color : 속성은 다단 레이아웃에서 각각의 컬럼 사이에 적용되는 선의 색상을 지정합니다.
  • column-rule-style : 속성은 다단 레이아웃에서 각각의 컬럼 사이에 적용되는 선의 스타일을 지정합니다.
  • column-rule-width : 속성은 단 레이아웃에서 각각의 컬럼 사이에 적용되는 선의 너비를 지정합니다.
  • column-span : 속성은 해당 열 요소를 다른 열 요소와 병합하여 하나의 큰 열로 만들어주는 역할을 합니다.
  • column-width : 속성은 다단 레이아웃에서 각 열의 너비를 지정하는 데 사용됩니다.
  • columns : 속성은 한 열(column)에 들어갈 요소의 크기와 수를 조절하여 여러 개의 열로 레이아웃을 만듭니다.
  • content : 속성은 가상 요소(pseudo-element)에 컨텐츠를 추가할 때 사용됩니다.
  • counter-increment : 속성은 카운터(counter) 기능을 사용할 때, 카운터의 값을 증가시키는 속성입니다.
  • counter-reset : 속성은 카운터를 초기화하는 속성입니다.
  • cursor : 속성은 마우스 커서의 모양을 지정할 수 있습니다.
  • direction : 속성은 텍스트의 쓰기 방향을 지정하는 데 사용됩니다.
  • display : 속성은 요소의 표시 방법을 지정하는 데 사용됩니다.
  • empty-cells : 속성은 빈 셀의 표시 여부를 지정하는 데 사용됩니다.
  • filter : 속성은 요소에 그래픽 효과를 적용하는 데 사용됩니다.
  • flex : 속성은 요소의 유연한 박스 모델(flexible box model)을 지정하는 데 사용됩니다.
  • flex-basis : 속성은 플렉스 항목(flex item)의 초기 크기를 설정합니다. 기본값은 'auto'입니다.
  • flex-direction : 속성은 플렉스 컨테이너(flex container) 내에서 플렉스 항목(flex item)의 배치 방향을 설정합니다.
  • flex-flow : 속성은 flex-direction과 flex-wrap 속성을 결합한 축약형 속성입니다.
  • flex-grow : 속성은 플렉스 아이템의 증가 가능한 크기를 지정하는 데 사용됩니다.
  • flex-shrink : 속성은 flex 요소의 크기를 줄일 수 있는 정도를 지정합니다.
  • flex-wrap : 속성은 flex 컨테이너 안에 있는 flex 아이템들의 줄바꿈 여부를 결정합니다.
  • float : 속성은 요소를 왼쪽이나 오른쪽으로 띄우는 데 사용되는 속성입니다.
  • font : 속성은 텍스트의 글꼴, 글꼴 크기, 글꼴 두께, 스타일 등을 지정하는 데 사용됩니다.
  • font-family : 속성은 폰트 종류를 설정합니다.
  • font-size : 속성은 폰트 사이즈 속성을 설정합니다.
  • font-size-adjust : 속성은 소문자를 기준으로 폰트 크기를 설정합니다.
  • font-stretch : 속성은 글꼴의 너비를 조절하는 데 사용됩니다. 이 속성은 특정한 글꼴 패밀리 내에서만 동작합니다.
  • font-style : 속성은 폰트의 스타일을 설정합니다.
  • font-variant : 속성은 글꼴 변형 형태를 일괄적으로 설정합니다.
  • font-variant-numeric : 속성은 숫자에 대한 글꼴 글리프(표시 형태)를 설정합니다.
  • font-variant-ligatures : 속성은 글꼴 글리프가 겹쳤을 때 설정합니다.
  • font-variant-emoji : 속성은 이모지 글꼴 글리프를 설정합니다.
  • font-variant-east-asian : 속성은 아시아 언어 글꼴 글리프를 설정합니다.
  • font-variant-caps : 속성은 소문자/대문자에 대한 글꼴 글리프를 설정합니다.
  • font-variant-alternates : 속성은 글꼴 글리프의 대체 형태를 설정합니다.
  • font-variant-position : 속성은 글꼴 글리프의 위치를 설정합니다.
  • font-variation-settings : 속성은 가변 폰트를 설정합니다.
  • font-weight : 속성은 폰트의 두께를 설정합니다.
