vue

Vue.js는 사용자 인터페이스를 구축하기 위한 progressiv한 JavaScript 프레임워크입니다
개발자가 웹 애플리케이션의 사용자 인터페이스를 개발하고 관리하는 데 도움을 주는 동적인 프런트엔드 프레임워크입니다.

Vue.js는 개발자에게 직관적인 문법과 유연한 기능을 제공하여 웹 애플리케이션의 개발을 용이하게 만듭니다. 또한 Vue.js는 다른 프레임워크와의 통합이 쉬우며, 기존 프로젝트에 점진적으로 도입하기도 쉽습니다.

vite

Vite는 Vue.js 및 기타 JavaScript 프레임워크를 위한 웹 개발 빌드 도구입니다. Vite는 빠른 개발 서버와 실시간 모듈 리로딩 기능을 제공하여 개발자가 빠르게 웹 애플리케이션을 개발하고 테스트할 수 있도록 도와줍니다.
Vite는 기존의 번들러와는 다른 동작 방식을 가지고 있습니다. 일반적으로 웹팩과 같은 번들러는 모든 모듈을 하나의 번들 파일로 묶은 후에 애플리케이션을 실행합니다. 하지만 Vite는 개발 시간을 단축하기 위해 빌드 프로세스에서 번들링하지 않고, 모듈을 개별 파일로 유지합니다. 이는 개발 서버가 애플리케이션의 모듈을 실시간으로 처리하여 빠른 로딩 속도와 즉각적인 변화를 제공할 수 있게 해줍니다.

Vite는 Vue.js 프레임워크에서 주로 사용되지만, 기타 프레임워크 및 라이브러리 (React, Preact, Svelte 등)와 함께도 사용할 수 있습니다. Vite는 기본적으로 ES 모듈 (ESM)을 사용하며, 최신 JavaScript 기능 (import/export 문법 등)을 지원합니다. 또한, 미리 컴파일된 빌트인 라이브러리를 제공하여 개발자가 추가적인 설정 없이 프레임워크 기능을 사용할 수 있도록 합니다.
